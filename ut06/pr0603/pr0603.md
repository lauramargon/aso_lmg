# PR0603: Manipulación de cadenas

En cada uno de los ejercicios se indica la cadena de entrada y la salida esperada. Debes indicar el comando o comandos necesarios para transformar la entrada en la salida.



## Ejercicio 1: Limpieza básica (mayúsculas, minúsculas y espacios)

**1.** Normalizar a mayúsculas

  * **Entrada:** `$sys = "Windows Server"`
  * **Salida:** `WINDOWS SERVER`
```bash
PS C:\Users\Administrador> $sys = "Windows Server"
PS C:\Users\Administrador> $sys.ToUpper()
WINDOWS SERVER
```
**2.** Normalizar a minúsculas

  * **Entrada:** `$user = "ADMINISTRATOR"`
  * **Salida:** `administrator`

```bash
PS C:\Users\Administrador> $user = "ADMINISTRATOR"
PS C:\Users\Administrador> $user.ToLower()
administrator
```

**3.** Limpieza de espacios extremos

  * **Entrada:** `$ip = " 192.168.1.10 "`
  * **Salida:** `192.168.1.10`

```bash
PS C:\Users\Administrador> $ip = " 192.168.1.10 "
PS C:\Users\Administrador> $ip.Trim()
192.168.1.10
```

**4.** Eliminación total de espacios

  * **Entrada:** `$mac = "00 AA 11 BB 22 CC"`
  * **Salida:** `00AA11BB22CC`

```bash
PS C:\Users\Administrador> $mac = "00 AA 11 BB 22 CC"
PS C:\Users\Administrador> $mac.Replace(" ", "")
00AA11BB22CC
```

**5.** Longitud de la cadena

  * **Entrada:** `$pass = "P@ssw0rd123"`
  * **Salida:** `11`

```bash
PS C:\Users\Administrador> $pass = "P@ssw0rd123"
PS C:\Users\Administrador> $pass.Length
11
```

## Ejercicio 5: Aplicaciones prácticas

**21.** Detección de usuario Admin (Boolean). Devuelve `True` si empieza por "ADM", `False` si no.

  * **Entrada:** `$u = "ADM_Lopez"`
  * **Salida:** `True`

```bash
PS C:\Users\Administrador> $u = "ADM_Lopez"
PS C:\Users\Administrador> $u -like "ADM*"
True
```

**22.** Formato de nombre propio. Convierte la primera letra en mayúscula y el resto en minúscula.

  * **Entrada:** `$nombre = "JAVIER"`
  * **Salida:** `Javier`

```bash
PS C:\Users\Administrador> $nombre = "JAVIER"
PS C:\Users\Administrador> ([string]$nombre[0]).ToUpper() + ($nombre.ToLower()).Substring(1)
Javier
```

**23.** Limpieza de Distinguished Name (AD). Extrae el nombre común (CN).

  * **Entrada:** `$dn = "CN=Beatriz,OU=Ventas,DC=dominio,DC=local"`
  * **Salida:** `Beatriz`

```bash
PS C:\Users\Administrador> $dn = "CN=Beatriz,OU=Ventas,DC=dominio,DC=local"
PS C:\Users\Administrador> $dn.Split(',')[0].Replace("CN=", "")
Beatriz
```

**24.** Generador de iniciales

  * **Entrada:**
    ```powershell
    $n = "Fernando"
    $a = "Alonso"
    ```
  * **Salida:** `F.A.`


```bash
PS C:\Users\Administrador> $n = "Fernando"
PS C:\Users\Administrador> $a = "Alonso"
PS C:\Users\Administrador> $n[0] + "." + $a[0] + "."
F.A.
```

**25.** Inversión de fecha. Transforma formato europeo a formato base de datos ISO (YYYYMMDD) sin guiones.

  * **Entrada:** `$euDate = "31-12-2023"`
  * **Salida:** `20231231`

```bash
PS C:\Users\Administrador> $euDate = "31-12-2023"
PS C:\Users\Administrador> [datetime]::ParseExact($euDate, 'dd-MM-yyyy', $null).ToString('yyyyMMdd')
20231231
```
