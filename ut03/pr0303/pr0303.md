### 6. Suma de dígitos

Pide un número al usuario y usa while para sumar todos sus dígitos. Por ejemplo, 123 debería devolver 6.
```bash
#!/bin/bash

suma=0
while [ $num -gt 0 ]
do
    digito=$((num % 10))
    suma=$((suma + digito))
    num=$((num / 10))
done
echo "La suma de los dígitos es: $suma"
```

### 10. Verificar contraseña

Usa until para pedir al usuario que ingrese una contraseña correcta, y repite hasta que la acierte.
```bash
#!/bin/bash
contraseña="secreto"
entrada=""
until [ "$entrada" == "$contraseña" ]
do
    read -sp "Introduce la contraseña: " entrada
    echo
    if [ "$entrada" != "$contraseña" ]; then
        echo "Contraseña incorrecta. Intenta de nuevo."
    fi
done
echo "Contraseña correcta. Acceso concedido."
```

### 13. Promedio de números ingresados

Usa while para pedir números al usuario hasta que ingrese "fin", luego muestra el promedio.
```bash
#!/bin/bash
suma=0
contador=0

while true
do
    read -p "Introduce un número (o 'fin' para terminar): " num
    if [ "$num" == "fin" ]; then
        break
    fi
    suma=$((suma + num))
    contador=$((contador + 1))
done

if [ $contador -gt 0 ]; then
    promedio=$((suma / contador))
    echo "El promedio es: $promedio"
else
    echo "No se introdujeron números."
fi
```

### 15. Juego de adivinar con límites dinámicos

```bash
        echo "Demasiado bajo."
    elif [ $intento -gt $num ]; then

    else
        echo "¡Correcto! El número era $num"
        break
    fi
```
